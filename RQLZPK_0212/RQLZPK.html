GitHub Parancsok

Repository létrehozása

git init - Lokális git repository inicializálása
git clone URL - Meglévő repository klónozása

Módosítások kezelése

git add fájlnév - Fájl hozzáadása a staging area-hoz
git commit -m "Üzenet" - Staged változások commitolása

Branch műveletek

git branch - Branch-ek listázása
git branch név - Új branch létrehozása
git checkout név - Branch váltás

Pull & Push

git pull - Frissítés a távoli repository-ből
git push origin branchNév - Lokális változások feltöltése a távoli repository-ba

Egyéb

git status - Státusz megjelenítése
git log - Commitok naplózása
