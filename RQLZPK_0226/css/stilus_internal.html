Internal stílus

Az "internal CSS" (belső stíluslap) azt a módszert jelenti, amikor a CSS kód közvetlenül az HTML dokumentumon belül található, a `style` elem segítségével a `head` részben. Ezzel a technikával a stílusokat egyetlen HTML fájlban tartjuk meg, így könnyű karbantartani kisebb projekteknél. A stílusokat a HTML kódban a megfelelő elemekre vonatkozóan adjuk meg, például a háttérszín, betűtípus vagy szövegszín beállítására. Az internal CSS segítségével könnyen definiálhatók és alkalmazhatók a formázási szabályok az adott dokumentumon belül.
